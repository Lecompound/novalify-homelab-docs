## Key Findings

> **Strengths:** VLAN segmentation, UniFi hardening, Tailscale mesh, ASUS firewall configuration.

> **Critical Gaps:** No local DNS, Proxmox LAN-offline, no backup verification, Frigate on management VLAN, MQTT unencrypted.

> **Immediate Priority:** Document recovery procedures, deploy local DNS, verify backup integrity.

## Infrastructure Status

Currently in transition — the EliteDesk-to-OPNsense migration is in progress, replacing the ASUS EBG19P as the primary gateway. An NVR watchdog crash occurred on 2026-06-30 (root cause: HiSilicon watchdog, not compromise).

Proxmox (192.168.50.190) is reachable via Tailscale but offline on the LAN. Headscale, Mac mini, and CE Link nodes are also offline — likely mid-migration state.
