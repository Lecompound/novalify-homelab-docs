# Network Services

## DHCP

- **Server:** ASUS EBG19P (192.168.50.1)
- **Scopes:** All four active VLANs
- **DNS assigned:** Cloudflare 1.1.1.2 / 1.0.0.2 (family-filtered)

## DNS

- Primary: ASUS port 53 → Cloudflare forwarders
- **No local DNS** — `pve.ebg`, `ha.ebg` etc. all NXDOMAIN
- This is a **critical gap** — all services rely on static IPs
