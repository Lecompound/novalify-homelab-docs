# Security Review

## Overall Posture: C+ (Developing)

Assessed across 8 domains against enterprise-grade controls adapted for the small-network context.

**Strengths:** VLAN segmentation, UniFi switch hardening, Tailscale mesh, ASUS firewall configuration.

**Critical Gaps:**
- No local DNS resolution
- Proxmox LAN-offline (Tailscale only)
- No verified backup integrity
- Frigate on management VLAN (camera traffic bypasses isolation)
- MQTT unencrypted on management VLAN
