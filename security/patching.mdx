## Gaps

### Hikvision CVE Risk — HIGH
Hikvision devices have known remote code execution vulnerabilities (CVE-2021-36260, others). The NVR is on an isolated VLAN which mitigates but does not eliminate the risk.

### No Automated Patching — MEDIUM
No unattended-upgrades or documented patch schedule. Systems may run for extended periods without security updates.

### No CVE Monitoring — MEDIUM
No routine checks for product CVEs affecting Hikvision, UniFi, or ASUS hardware.

### No Firmware Baseline — MEDIUM
Current firmware versions are not recorded or tracked.

## Recommendations

1. Establish a **monthly patch cadence** with documented changelog
2. Monitor **Hikvision PSIRT** and **Ubiquiti release notes**
3. Subscribe to **CVE feeds** for all in-use software
4. Document a **firmware baseline** for each device
5. Implement **automatic security updates** on Debian/Ubuntu hosts
