# 6. Ongoing Procedures

## Monthly Patch Cadence

First Monday of each month:

```bash
# Check available updates
ssh root@192.168.50.145 "apt update && apt list --upgradable"
ssh root@100.89.90.38 "apt update && pveupdate && pveupgrade"

# Apply and reboot if kernel update
ssh root@192.168.50.145 "apt upgrade -y && systemctl reboot"
```

## Quarterly Security Checklist

- [ ] All firmware versions current
- [ ] No new CVEs for in-use software
- [ ] SSH key-only enforced on all hosts
- [ ] No unexpected open ports
- [ ] Backup integrity verified
- [ ] Secrets rotation done (90 days)
- [ ] IR tabletop exercise completed
