## Architecture Notes

- **No subnet router** — Tailscale does not bridge to any LAN subnet
- **Identity-based access** — all nodes authenticate via Tailscale ACLs
- **No LAN bridging** — the tailnet preserves VLAN isolation at Layer 2
- **Client version mismatch** — current client 1.98.5 vs tailscaled 1.98.5 (minor variant, non-critical)
