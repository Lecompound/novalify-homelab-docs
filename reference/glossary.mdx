# Glossary

| Term | Definition |
|------|-----------|
| ASUSWRT | ASUS router firmware platform |
| CGNAT | Carrier-Grade NAT — Telstra shares one public IP among subscribers |
| Cloudflare Tunnel | Encrypted tunnel from `miniapp.lecompound.us` → local service |
| Frigate | Open-source NVR software with AI object detection |
| LXC | Linux Container (Proxmox container type) |
| L2/L3 | Layer 2 (data link) / Layer 3 (network) — OSI model layers |
| MCP | Model Context Protocol — standard for AI tool integration |
| MGMT | Management VLAN — trusted infrastructure network |
| MQTT | Lightweight pub/sub messaging protocol (used by IoT/HA) |
| Nabu Casa | Cloud service for Home Assistant remote access |
| NCP | Network Controller Platform — UniFi controller host |
| NVR | Network Video Recorder |
| OPNsense | Open-source firewall platform (planned migration target) |
| PVE | Proxmox Virtual Environment |
| RSTP | Rapid Spanning Tree Protocol |
| RTSP | Real-Time Streaming Protocol (used by cameras) |
| Samba | SMB/CIFS file sharing implementation |
| Suricata | Open-source IDS/IPS engine |
| Tailscale | WireGuard-based mesh VPN with identity-aware access |
| Unbound | Validating, recursive DNS resolver |
| UniFi | Ubiquiti's managed networking platform |
| VLAN | Virtual Local Area Network — logical network segment |
| VM/VMID | Virtual Machine / Virtual Machine ID (Proxmox) |
